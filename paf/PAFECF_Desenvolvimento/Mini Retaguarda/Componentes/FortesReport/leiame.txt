INSTALAÇÃO
----------

A instalação deverá ser feita manualmente através da cópia dos arquivos para
a pasta (DELPHI)\Projects\Bpl. Se desejar instalar em outra pasta, esta
deverá ser adicionada à lista de diretórios de busca do Delphi
(Menu: Tools - Environment Options - Library - Library path)
Em seguida, adicione o pacote .bpl ao Delphi
(Menu: Component - Install Packages...)
